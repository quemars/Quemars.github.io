# Global Time App
## Project goal
Build a simple and friendly app that;
- Displays local time for a chosen city and updates every second
- Allows selecting one city from a dropdown 
- Enhanced Feature (idea): allow adding/removing multiple cities and a dark/light theme
- Advanced Feature (idea): timezone event conversion and world map with clickable cities

## Repo setup
This repo holds the source for the Global Time App. Currently setting up the repo and project tasks no site files are published yet.
